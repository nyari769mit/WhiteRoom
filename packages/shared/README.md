# @whiteroom/shared

Shared types, constants, and utilities for the WhiteRoom governance engine.

## Install

```bash
npm install @whiteroom/shared
```

## Contents

- **Types** — `AgentState`, `FleetConfig`, `GovernanceDecision`, `AuditEvent`, `Workspace`, etc.
- **Constants** — `DEFAULT_WATCH_LIMIT_TOKENS`, `DEFAULT_REST_SECONDS`, `EVENT_TYPES`
- **Pricing** — `MODEL_PRICING`, `calculateCostCents()`, `TIER_MODEL_MAP`
- **ID generation** — `generateId()`, `generateTrialToken()`, `generateApiKey()`
- **Compression** — `COMPRESSION_SYSTEM_PROMPT`, `buildCompressionMessages()`
